<h2 id='installation'>Installation</h2>

```bash
** open terminal
git clone https://github.com/shd-mn/github-info.git
cd github-info

** then
** install packages
pnpm install
```

<h2 id='starting'>Starting Project</h2>

```bash
pnpm dev

** open web page in browser
http://localhost:3000
```
